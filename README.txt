1. Рекомендации:


	1.1 Для того, чтобы программа работала корректно, нужно поместить .exe-файл "Painting Program" в одну папку с папкой "icons", 
содержащей изображения, используемые для интерфейса программы.


	1.2 В целях корректной работы программы, не рекомендуется переименовывать изображения в папке "icons", так как есть риск 
отсутствия видимого отображения иконок в приложении Painting Program вследствие необнаружения программой искомых файлов.

2. Инструкция по эксплуатации:

	2.1 Выбор Инструмента
	Наша программа обладает функционалом, позволяющим создавать графические рисунки. Инструменты, позволяющие изменять рисунки находятся 
во вкладке меню "Инструмент". В их число входят:


		2.1.1 Кисть - стандартная кисть, позволяющая рисовать кривые линии. При зажатии ЛКМ (Левой Кнопки Мыши), программа начинает 
процесс рисования. Кисть будет двигаться и рисовать, следуя за Вашим курсором. Конец процесса рисования наступит после отпускания ЛКМ.
Кисть обладает следующим набором параметров:

			2.1.1.1 Размер. Пользователь может изменять толщину кисти, выбрав нужный размер в разделе меню "Толщина". В приложении
есть предустановленные параметры толщины: 4px, 7px, 9px, 12px, 19px (px - количество пикселей). Также пользователь может указать свой
предпочитаемый размер кисти, нажав на кнопку "Своя". После нажатия вылезет всплывающее окно со строчкой ввода желаемого размера кисти.

			2.1.1.2 Цвет. Пользователь может изменять цвет кисти, выбрав нужный цвет в разделе меню "Цвет". В приложении присутствуют
предустановленные параметры цвета: красный, оранжевый, желтый, зеленый, голубой, синий, фиолетовый, черный, белый. Также пользователь 
может указать свой предпочитаемый цвет кисти, нажав на кнопку "Другой". После нажатия вылезет всплывающее окно с доступным действием 
выбора желаемого цвета кисти


		2.1.2 Ластик - инструмент, позволяющий стирать нарисованное другими инструментами. При зажатии ЛКМ, программа начинает процесс
рисования. Ластик будет двигаться и стирать, следуя за Вашим курсором. Конец процесса стирания наступит после отпускания ЛКМ.
Ластик обладает следующим набором параметров:

			2.1.2.1 Размер. Пользователь может изменять толщину ластика. Для этого надо выбрать инструмент "Ластик". В появившемся
субменю есть возможность выбрать размер. В приложении есть предустановленные параметры толщины: 4px, 7px, 9px, 12px, 19px. Также 
пользователь может указать свой предпочитаемый размер ластика, нажав на кнопку "Свой". После нажатия вылезет всплывающее окно со строчкой
ввода желаемого размера кисти.


		2.1.3 Прямая линия - инструмент, позволяющий пользователю рисовать прямые линии. При выборе этого режима, открывается 
субменю, позволяющее пользователю выбрать тот стиль, которым он хочет нарисовать прямую. В программе есть предустановленные стили: цельная,
отрывистая, точечная. При выборе одного из стилей, пользователь будет рисовать прямую выбранным стилем, до момента следующего выбора 
стиля либо смены инструмента. Процесс рисования прямой начинается с момента зажатия ЛКМ. На позиции начала рисования ставится точка,
а вся прямая будет изображена лишь после того, как пользователь отпустит ЛКМ. Прямая будет проведена из точки зажатия ЛКМ до точки
отпускания ЛКМ. Помимо изменения стиля рисования прямой линии, этот инструмент обладает следующим набором параметров:

			2.1.3.1 Размер. Пользователь может изменять толщину прямой, выбрав нужный размер в разделе меню "Толщина". В приложении
есть предустановленные параметры толщины: 4px, 7px, 9px, 12px, 19px (px - количество пикселей). Также пользователь может указать свой
предпочитаемый размер кпрямой, нажав на кнопку "Своя". После нажатия вылезет всплывающее окно со строчкой ввода желаемой толщины прямой.

			2.1.3.2 Цвет. Пользователь может изменять цвет прямой, выбрав нужный цвет в разделе меню "Цвет". В приложении присутствуют
предустановленные параметры цвета: красный, оранжевый, желтый, зеленый, голубой, синий, фиолетовый, черный, белый. Также пользователь может указать
свой предпочитаемый цвет прямой, нажав на кнопку "Другой". После нажатия вылезет всплывающее окно с доступным действием выбора желаемого цвета прямой.



	2.2 Параметры рисования
	В процессе рисования либо редактирования рисунка, пользователь может изменять некоторые параметры приложения. Ниже приведены те, параметры,
что может изменять пользователь в данном разделе


		2.2.1 Цвет фона - инструмент, с помощью которого пользователь может изменять цвет основного фона приложения. Чтобы его изменить,
пользователю нужно нажать на панели меню на вкладку "Параметры", и в появившемся меню выбрать режим "Цвет фона". После нажатия вылезет всплывающее
окно с доступным действием выбора желаемого цвета фона приложения.


		2.2.2 Цвет панели меню - инструмент, с помощью которого пользователь можжет изменять цвет панели меню. Чтобы его изменить, пользователю
нужно нажать на панели меню на вкладку "Параметры", и в появившемся меню выбрать режим "Цвет панели меню". После нажатия вылезет всплывающее окно с
доступным действием выбора желаемого цвета панели меню приложения.



	2.3 Текущий инструмент
	Может возникнуть сложность в том, что пзователь забыл, каким инструментом он пользуется в данный момент. Для предотвращения этой неприятности,
на панели меню нашей программы есть раздел "Текущий инструмент". При нажатии на него пользователь увидет название того инструмента, который он сейчас
использует. При нажатии на этот раздел сразу после запуска программы, пользователю ничего отображено не будет.



	2.4 Опереции над файлом
	Наша программа расчитана не на одноразовый процесс рисования. Это значит, что пользователь может производить определенные операции над текущим файлом,
а также открывать уже готовые проекты. Для этого у нас есть определённый перечень инструментов:
	
		
		2.4.1 Сохранить лист - операция, позволяющая пользователю сохранить свой проект на устройство. После выбора в разделе "Файл" операции "Сохранить
лист", вылезет всплывающее окно с доступным действием сохранения проекта в директорию устройства.
		

		2.4.2 Очистить лист - операция, позволяющая пользователю привести холст к изначальному состоянию. После выбора в разделе "Файл" операции "Очистить
лист", основной фон и панель меню станет белого цвета, также произойдут некоторые изменения в выборе инструмента. Если перед очисткой листа последним инструментом
была ручка, то ничего не изменяется, но если нет, то инструментом становится ручка, с ранее выбранными параметрами.


		2.4.3 Открыть - операция, позволяющая пользователю открыть фотографию, либо рисунок. Важно понимать, что разрешение открываемого файла не может 
превышать разрешение устройства пользователя. После выбора в разделе "Файл" операции "Открыт", вылезет всплывающее окно с доступным действием открытия проекта из
директории устройства. При попытке пользователя открыть файл, разрешением больше разрешения устройства, вылезет соответствующая ошибка. 


		2.4.4 Выйти - операция, позволяющая закрыть приложение. Важно понимать, что без предварительного сохранения проекта, все изменения с закрытием
будут сброшены. После выбора в разделе "Файл" операции "", вылезет всплывающее окно с доступным действием сохранения проекта в директорию устройства.
